Visualisation of various Sorting Algorithms using "ncurses" library

	1: Bubble sort
	2: Selection sort
	3: Insertion sort
	4: Merge sort
	5: Quick sort
______________________________________________
_______________________
Installation steps-

1. Goto the VisualisationSort/ directory
2. Execute the cmake command
	cmake .
3. Make the repo
	make
______________________________________________________________________
Bugs-

1. Error at graph-utils/menu.c source code (Fatal)
______________________________________________________________________
Check test/* directory (Bug-free)

Compile and run standalone programs
	gcc bsort.c -lncurses
	./a.out
